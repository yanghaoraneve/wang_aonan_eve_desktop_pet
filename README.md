# 桌宠小楠

桌宠小楠是一款以王澳楠 EVE 为灵感制作的 Q 版桌面宠物应用。它可以常驻在桌面上，支持换装、动作动画、聊天 Agent、记忆、歌词知识库、日程与提醒等功能。

## 下载安装

| 系统 | 安装包 | 说明 |
| --- | --- | --- |

## 主要功能

- **桌面宠物**：透明悬浮窗口，支持待机、挥手、跳跃、跑动、思考等动画状态。
- **三套装扮**：内置红白裙、黑色皮革发带、竹笛棕色造型，可在设置或菜单中切换。
- **聊天 Agent**：支持 OpenAI 兼容接口，可配置 DeepSeek 等模型服务。
- **长期记忆**：聊天过程中会提取稳定偏好和重要信息，用于后续对话上下文。
- **歌词知识库**：内置王澳楠 EVE 相关歌词资料，聊天时可作为本地知识来源。
- **日程提醒**：可通过聊天创建提醒，并在触发时弹窗提示。
- **桌面小聊**：可在桌宠附近显示轻量输入框和最新回复气泡。

## 使用方法

1. 下载并安装对应系统的安装包。
2. 启动“桌宠小楠”。
3. 右键桌宠打开菜单，可进入聊天、设置、换装和动作列表。
4. 在设置中填写模型服务地址、模型名称和 API Key。
5. 回到聊天窗口，即可和小楠对话、创建提醒或查看日程。

## 模型配置

本应用使用 OpenAI 兼容格式请求模型服务。常见配置包括：

- API Base URL：模型服务的接口地址
- Model：模型名称
- API Key：模型服务密钥

API Key 会保存在本机应用配置中，不会提交到仓库。

## 源码说明

源码位于 [`app/`](./app/) 目录，技术栈为：

- Tauri 2
- Rust
- TypeScript
- Preact
- SQLite

本地开发：

```bash
cd app
npm install
npm run tauri:dev
```

本地打包：

```bash
cd app
npm install
npm run tauri:build
```
